  <action>
1. Install the Catppuccin Tailwind CSS plugin:
```bash
npm install @catppuccin/tailwindcss@1.0.0-beta.1
```

2. Update src/index.css to import Catppuccin Mocha theme.

Add after the tailwindcss import (but before font imports if they exist):
```css
@import "@catppuccin/tailwindcss/mocha.css";
```

The final import order should be:
```css
@import "tailwindcss";
@import "@catppuccin/tailwindcss/mocha.css";
/* fonts if present */
```

This exposes all Catppuccin Mocha colors as ctp-* utilities:
- ctp-base (dark background) - replaces gray-950
- ctp-mantle (slightly lighter) - replaces gray-900
- ctp-surface0 (surface) - replaces gray-800
- ctp-surface1 (elevated surface) - replaces gray-700
- ctp-text (primary text) - replaces gray-100/white
- ctp-subtext0/1 (muted text) - replaces gray-400/500
- ctp-blue, ctp-green, ctp-red, ctp-peach, etc. (accent colors)
  </action>
  <verify>
Run `npm run tauri dev`. In DevTools, add a test class `bg-ctp-blue` to any element - it should render as Catppuccin blue (#89b4fa).
  </verify>

Replace gray-* with ctp-* in buttonVariants:
```tsx
const buttonVariants = cva(
  "inline-flex items-center justify-center whitespace-nowrap rounded-md text-sm font-medium transition-colors focus-visible:outline-none focus-visible:ring-1 focus-visible:ring-ctp-overlay0 disabled:pointer-events-none disabled:opacity-50",

    variants: {
      variant: {
        default: "bg-ctp-blue text-ctp-base shadow hover:bg-ctp-blue/90",
        destructive: "bg-ctp-red text-ctp-base shadow-sm hover:bg-ctp-red/90",
        outline:
          "border border-ctp-surface1 bg-transparent shadow-sm hover:bg-ctp-surface0 hover:text-ctp-text",
        secondary: "bg-ctp-surface1 text-ctp-text shadow-sm hover:bg-ctp-surface0",
        ghost: "hover:bg-ctp-surface0 hover:text-ctp-text",
        link: "text-ctp-blue underline-offset-4 hover:underline",
      },
      // ... size variants unchanged
    },
  },
);
```

**input.tsx:**
Replace gray-* with ctp-*:
- bg-gray-800 -> bg-ctp-surface0
- border-gray-700 -> border-ctp-surface1
- text-gray-400 (placeholder) -> text-ctp-overlay0
- focus:border-blue-500 -> focus:border-ctp-blue
- focus:ring-blue-500 -> focus:ring-ctp-blue

**dialog.tsx:**
Replace gray-* with ctp-*:
- bg-gray-900 -> bg-ctp-mantle
- border-gray-800 -> border-ctp-surface0
