# Milestone v1.2.0: Bugfixing & Polish

**Status:** SHIPPED 2026-02-07
**Phases:** 16-19
**Total Plans:** 17

## Overview

v1.2.0 addresses accumulated UX bugs, visual inconsistencies, and adds a command palette with registry architecture. Starting with quick visual fixes and bug squashing, then improving the hierarchical view and conventional commit visuals, building the command palette as the milestone's most architecturally significant feature, and finishing with settings expansion, onboarding, and file icons.

## Key Accomplishments

1. **Command Palette** — VS Code-style command palette with registry pattern, fuzzy search, ARIA combobox semantics, and 14 registered commands
2. **Settings Expansion** — Git identity (name, email, default branch) and Integrations (editor, terminal) tabs with platform-aware options
3. **Git Init Onboarding** — Non-git folder detection with initialization prompt and default branch configuration
4. **Folder Staging** — Batch stage/unstage entire folders in hierarchical view with consistent icon spacing
5. **Colored Commit Types** — Shared COMMIT_TYPE_THEME module with colored icons across topology, history, changelog, and commit form
6. **Visual Polish** — Modal flicker fix, subtler blade animation, tag sorting, stash labels, diff header formatting, blade reset on repo switch
