# Quick Task 007: Dark Theme for React Flow Controls

## Objective

Apply dark theme styling to React Flow controls panel to match the application's dark theme.

## Task

Add CSS overrides to `src/index.css` to style the React Flow controls with dark colors:

- Background: gray-950 (#030712)
- Borders: gray-800 (#1f2937)

## Success Criteria

- React Flow controls blend with dark theme
