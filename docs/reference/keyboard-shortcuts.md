# Keyboard Shortcuts

FlowForge supports keyboard shortcuts for common operations. On macOS, **Cmd** is used; on Windows and Linux, **Ctrl** is used instead.

## General

| Shortcut | Action |
| --- | --- |
| `Cmd/Ctrl + O` | Open repository |
| `Cmd/Ctrl + ,` | Open settings |
| `Cmd/Ctrl + Shift + P` | Open command palette |
| `Escape` | Close current blade / panel |
| `Backspace` | Navigate back (pop blade) |

## Git Operations

| Shortcut | Action |
| --- | --- |
| `Cmd/Ctrl + Shift + A` | Stage all files |
| `Cmd/Ctrl + Shift + U` | Push to remote (Upload) |
| `Cmd/Ctrl + Shift + L` | Pull from remote (Latest) |
| `Cmd/Ctrl + Shift + F` | Fetch from remote |
| `Cmd/Ctrl + Shift + M` | Toggle amend commit mode |

## Topology View

| Shortcut | Action |
| --- | --- |
| `Enter` | Open details for selected commit |
